# Minecraft
매타버스아카데미 1차 융합 게임 모작 프로젝트 VRsing

## 프로젝트 소개
기획 1명, TA 2명, 개발 3명으로 구성된 팀이 협업하여 진행한 게임 모작 프로젝트

각 분야의 역할 분담과 효과적인 협업 방식을 익히는 것을 목표로 진행
